Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHtml means we write direct html code inside this, innerText and textContent(efficient than innerText) almost same and they are used for read/write texts inside element

Question-3: What is event delegation in the DOM?

a single event listener to a parent element instead of attaching event listeners to each individual child element.

Question-4: What is event bubbling in the DOM?

If we have an event on a child element, the event will first be executed on that element, and then it will bubble up to its parent, then its parent's parent, until it reaches the topmost parent.

Question-5: How do you create, add, and remove elements using JavaScript?
